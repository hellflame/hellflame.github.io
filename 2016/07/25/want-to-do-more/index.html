I Want to build more

发布 : 2016-07-25 分类 : diaries 浏览 : --

深圳半月

眼看今天就是来深圳半个月的时间点了捏～～～腾讯为新员工提供的半个月豪华酒店的住宿在今天一早也以退房结束了捏～～这也是这篇blog存在的原因捏～～

好了～～就以此纪念一下来腾讯实习的半个月好了啦～～

好吧～腾讯为新员工提供的暂住酒店的确挺好的，每天早上还有早餐(不过貌似不是所有实习生都这样)，(可能是由于这一时间段来实习的孩纸是奇数，又没有特别高级的入职员工，我恰好就一个人住在了一个标准间，为此应该也拉来了无数仇恨了吧23333真是优越的喵)过一段路之后还会有班车经过，对于刚到深圳的喵来说真的很贴心捏，毕竟想起刚到深圳时候没有太阳，找不到北的感觉，还是让班车老司机带我们到公司就好了23333，当然下班也是有班车经过的捏～～所以喵还不至于迷路

说来也巧，在这段实习期间经历了整个部门的半年会，除了老大总结半年工作之外，最重要的当然还是大家出去团建活动啦～

先不说去哪里，喵觉得住的酒店就很值的样子23333

好吧，果然还是别人拍的照片比较美2333

好吧，，其实我比较好奇他们是怎么能在外墙拍下这些的，，，难道是神奇的自拍杆么

酒店的夜景其实也是挺美的，只是晚上用手机拍照质量挺渣的

当然，那天晚上也体验了一回所谓的灯红酒绿的世界了吧，，好吧，只能承认自己已经喝的差不多了，深圳的啤酒也是啦么的醉人。这场晚宴中还有小鲜肉们的节目表演(同样为小鲜肉的喵不知道算是幸运还是不幸的错过了组委会找人排练节目的机会，在台下默默的当了观众)，无论表演怎样，和当初自己能预见的一样，这是一个节操不再的舞台，在舞台下静静的捡节操就好了

晚上的晚宴固然节操满天飞，然而白天的时候大家也是玩的很high的样子捏～～大部分人选择了在古龙峡漂流，对于不会水的喵来说，当然只能和少部分人选择瀑布探险了捏～～不过城市里怎么可能会有真正的探险(好吧，走过索道的时候喵的腿的确都软了23333)

仁者乐山，智者乐水，到了探险的时候，果然还是拍水比较美捏～～

虽然其实上一次和女王大人去青城山的时候也拍了很多水的说捏～～毕竟水在这大山深处也算是与众不同的存在了捏～～一种即有形又无形的存在，蕴藏着未知的深邃和秘密

不知道为什么，也不知道从什么时候开始，镜头中少了活着的事物，当然也不会让自己入境(毕竟不能直视23333)，也许是因为看着这些不会改变的生灵，会有一种莫名的平静出现在心中吧

心中活着的东西被逐渐替换为死的东西，最终自己也将随之死去，这会是喵的最终解决么？

虽然一整天写代码的日子还是挺辛苦的感觉，毕竟回到酒店之后都会觉得比较累，虽然有可能是班车比较累2333，但是随着每天忙碌的日子变得多了起来，心中的空虚感也在逐渐累积起来

在学校时不是用代码来填充这样的空虚感的么？然而到了公司每天写代码，为什么还会有空虚感捏～

公司与学校的区别

在公司与在学校写代码最大的区别也许就是其目的不同了吧。

公司的目的是要发展下去，要发展下去就需要更多的资金，所以整个公司最大的目的就是在于利润，在年中会上老大也是一语道破今年半年会的规格远超过曾经年终会的原因，因为我们有了更多的钱，大家写代码的最终目的，当然也是为了这个(好吧，虽然喵也知道有人真的是因为兴趣在coding，无论自己的职位的大神)

无论是使用过时但是无比稳定的老框架，还是刚刚写好的快速开发框架，或者是今天刚刚讲堂上宣讲的大型运维平台，大家都是为了更加高效的完成任务，这并没有什么不好，按照别人的需求，达到预期的效果，一切都是啦么的完美。恰好自己所在的team也使用了很多曾经甚至都没有听说的语言和框架，所以也学习到了很多东西

然而在学校的自由编程都是自己给自己提需求，满足自己的需要就好了，一切都会向着自己感兴趣的方向发展

其实想说的是，在公司和学校其实都挺好的，能够学习到很多东西。然而喵并不想马上就为了钱而奔波

然而作为喵而言，并不想像身边的同学一样，只是为了更加安稳的工作条件而留下来，为了丰厚的工资而留下来，喵并不是说这样的期望存在问题，只是喵自己并不想这么早就投入到所谓的生产社会中担任一颗螺丝钉，我还想要更加自由而广阔的发展空间，也许在别人眼里，喵的选择不过是为了推迟就业，然而喵只是想说，我跟你们不一样

想起刚刚从武汉到深圳的火车上下来的时候的热度，跟武汉是多么的相似～～没来得及注意火车站出站口的香港相关的特别标识，到了地铁站才发现，武汉其实已经算是内地了，深圳的服务对象，不仅有内地人，本地人，还有英语国家大众，第一次感受到了对外区域的纷繁复杂，还有，物价真的比武汉要贵，要是不是公司福利比较好，可能自己真的独自一人生存不下去吧，也再一次感受到不能什么都依靠其他人，可能也因此，喵会变得更加独立了吧～

关于接下来喵将要走的路～～也许喵已经在此刻作出了选择了吧～～

I Juts want to build more, more and more～～
